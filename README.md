# No Bus Harassment Mod / 公交车防骚扰模组

[English](#english) | [中文](#中文)

---

## English

This mod disables the passenger harassment, molestation, and delinquent events that a player might encounter when riding the bus in Degrees of Lewdity.

### Features
- **Disable regular bus harassment**: Prevents normal groper and gangbang events on the bus.
- **Disable pregnancy touch harassment**: Bypasses the option for passenger harassment when a pregnant player lets a passenger touch their belly.
- **Disable sleep/passout bus harassment**: Bypasses being molested or stripped when falling asleep or passing out on the bus.

### Settings
You can customize which functions are enabled in the game's **Options** settings page.

### Dependencies
- ModLoader (>= 2.3.1)
- TweeReplacer (>= 1.6.0)
- maplebirchFramework (>= 3.0.0)

## 中文

本模组关闭了玩家在《Degrees of Lewdity》中乘坐公交车时可能会遇到的乘客骚扰、非礼以及不良学生霸凌等事件。

### 功能介绍
- **关闭普通乘车骚扰**：防止公交车上普通的电车痴汉及引擎故障时的群殴/轮暴事件。
- **关闭怀孕触摸骚扰**：当怀孕玩家允许乘客触摸腹部时，屏蔽可能触发的非礼对话与后续事件。
- **关闭乘车起冲突/睡眠/晕倒骚扰**：防止在车上睡着或晕倒时被剥光衣服或非礼。

### 模组设置
您可以在游戏侧边栏的 **选项 (Options)** 页面中自由配置上述每项关闭功能是否生效。

### 依赖项
- ModLoader (>= 2.3.1)
- TweeReplacer (>= 1.6.0)
- maplebirchFramework (>= 3.0.0)
